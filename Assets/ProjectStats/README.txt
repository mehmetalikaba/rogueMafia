ProjectStats
=================

This tool will show tou statistics about your project and will generate a logfile with the information.

Installation
------------
1. Import the ProjectStats package into your Unity project.
2. Open the tool from the Unity menu: Window -> ProjectStats

Usage
-----
1. Click on the "Analyze" button to generate a logfile with the information.
2. Click on the "Open Log File" button to open the logfile.
